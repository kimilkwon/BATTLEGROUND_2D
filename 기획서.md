# 🎮 BATTLEGROUND 2D — 게임 기획서

> 브라우저 기반 탑다운 2D 배틀로얄 | 최대 8인 실시간 멀티플레이

---

## 1. 개요

| 항목 | 내용 |
|---|---|
| 장르 | 탑다운 2D 배틀로얄 |
| 플레이어 | 최대 8명 (솔로 / 듀오) |
| 판 시간 | 약 7~12분 |
| 플랫폼 | 브라우저 (PC 우선, 모바일 지원) |
| 시점 | 탑다운 (카메라는 내 캐릭터 중심) |

---

## 2. 기술 스택

```
클라이언트  : Phaser 3 (렌더링, 물리, 카메라, 타일맵)
서버        : Node.js + Colyseus (게임룸, 상태동기화, 서버 틱)
통신        : WebSocket (Colyseus 내장)
맵 제작     : Tiled Map Editor → Phaser 로드
배포        : Render (서버) + Vercel (클라이언트 정적)
틱 레이트   : 서버 20tick/s, 클라이언트 보간(interpolation)
```

---

## 3. 맵 설계

### 3-1. 맵 크기
- **전체 맵**: 4000 × 4000px (타일 32px 기준 → 125×125 타일)
- **카메라**: 플레이어 중심, 시야 반경 제한 (아래 참고)
- **줌**: 기본 1.5x, 조준경 장착 시 조절

### 3-2. 지형 구성
| 지형 | 설명 |
|---|---|
| 평야 | 기본 지형, 엄폐물 없음 → 위험 구역 |
| 숲 | 나무로 시야 차단, 아이템 스폰 많음 |
| 도시 | 건물 밀집, 실내전, 고급 아이템 스폰 |
| 군사기지 | 맵 중앙/외곽 1곳, 최고급 아이템 + 메카로봇 스폰 |
| 해안/물 | 이동 불가, 경계선 역할 |
| 폐허 | 파괴된 건물, 엄폐물로 활용 |

### 3-3. 파괴 가능 엄폐물
- 나무 상자, 드럼통, 나무 울타리 → 총격으로 파괴 가능
- 콘크리트 벽, 금속 컨테이너 → 불파괴, 단 총알 관통 불가
- 드럼통 파괴 시 **폭발 범위 데미지**

## 5. 무기 시스템

### 5-1. 무기 목록

| 무기 | 데미지 | 탄창 | 연사 | 사거리 | 레어도 |
|---|---|---|---|---|---|
| **권총** | 25 | 12 | 반자동 | 중 | ⬜ 일반 |
| **산탄총** | 80 (근거리) | 6 | 펌프 | 단거리 | 🟩 고급 |
| **라이플 (AR)** | 35 | 30 | 자동 | 중장거리 | 🟩 고급 |
| **저격총** | 120 | 5 | 볼트액션 | 장거리 | 🟦 희귀 |
| **SMG** | 20 | 45 | 자동 | 단중거리 | ⬜ 일반 |
| **LMG** | 28 | 100 | 자동(느린 준비) | 중거리 | 🟪 영웅 |

### 5-2. 반동 시스템
- 연사할수록 조준점이 위로 떠오르는 **스프레이 패턴** 적용
- 반동은 **마우스를 내리면서 쏘면 제어 가능** (스킬 요소)
- 무기별 반동 세기: SMG < 권총 < AR < LMG < 저격총(단발이지만 크게 밀림)
- 이동 중 사격 시 반동 1.5배

### 5-3. 조준경

| 조준경 | 시야 배율 | 추가 시야 반경 |
|---|---|---|
| 2배 | ×2 | +20% |
| 4배 | ×4 | +40% |
| 8배 | ×8 | +70% |
| 12배 | ×12 | +100% (저격총 전용) |

- 조준경 장착 중 우클릭으로 **조준 모드** 전환 → 이동 속도 -50%, 시야 확장

## 6. 아이템 시스템

| 아이템 | 회복량 | 사용 시간 | 특이사항 |
|---|---|---|---|

### 6-2. 방어구

**헬멧**

|---|---|---|

| 헬멧 2 | 머리 데미지 -50% | 100 |

**방탄조끼**

## 7. 메카 로봇 시스템

### 7-1. 기본 정보
- 맵에 **2기 고정 스폰** (군사기지 내부)
- 탑승 시 HP의 **50% 즉시 소모** (탑승 비용)
- 로봇 자체 HP: **300** (별도 체력바)
- 탑승 가능 인원: **1명**

### 7-2. 능력
| 능력 | 내용 |
|---|---|
| 미사일 연사 | 10초간 무제한 미사일 발사 (클릭 방향으로 발사) |
| 미사일 데미지 | 직격 70, 범위 폭발 30 |
| 이동속도 | 일반 캐릭터의 70% (둔함) |
| 방어력 | 총알 데미지 -40% |
| 시야 | 탑승 시 시야 반경 +30% |

### 7-3. 10초 후
- 로봇이 과열 → **자동 탈출** (플레이어는 남은 HP로 생존)
- 15초 쿨타임 후 재탑승 가능 (로봇 HP가 남아있으면)
- 로봇 HP 0 → **폭발 데미지** 반경 200px에 80 데미지

## 8. 시야 시스템

### 8-1. 기본 시야
- 캐릭터 중심으로 **원형 시야 반경 350px**
- 시야 밖은 **어둡게 처리** (완전한 블랙 아웃은 아님, 탐색은 가능)
- 시야 안의 적만 렌더링

### 8-2. 시야 변경 요소
| 상황 | 시야 반경 |
|---|---|
| 기본 | 350px |
| 이동 중 | 300px (주의력 분산) |
| 엎드리기 | 280px |
| 조준경 조준 모드 | 조준경 배율에 따라 최대 700px |

| 섬광탄 피격 | 화면 완전 하얗게 |

## 9. 추가 재미 요소

### 9-1. 에어드롭 ✈️
- 게임 중 **1~2회** 랜덤 위치에 에어드롭 투하
- 낙하 전 **30초 전 미니맵에 투하 위치 표시** → 누가 먼저 갈 것인가!
- 에어드롭 내용물: 레어 무기, 갑옷3, 힐팩 다수, 조준경 8배/12배

### 9-2. 차량 시스템 🚗
| 차량 | 탑승 인원 | 속도 | 특이사항 |
|---|---|---|---|
| 오토바이 | 1명 | 매우 빠름 | 조향 어려움, 로드킬 가능 |
| 지프 | 2명 | 빠름 | 총격 가능 (조수석) |
| 장갑차 | 1명 | 보통 | 총알 데미지 -60%, 느림 |

- 차량은 파괴 가능 (HP: 오토바이 150, 지프 300, 장갑차 500)
- 파괴 시 폭발

### 9-3. 날씨 이벤트 🌧️
- 게임 중 랜덤으로 날씨 변화 (2분 주기)
- **비**: 발소리 마스킹 (미니맵 음파 표시 안 됨)
- **안개**: 전체 시야 -30%
- **맑음**: 기본 상태

### 9-4. 헤드샷 시스템 💀
- 머리 판정: 데미지 ×2
- 헬멧이 없으면 저격총 1방 즉사
- 헤드샷 시 화면에 **"HEADSHOT!"** 연출

### 9-5. 녹다운 & 부활 시스템 💉
- HP 0 → 즉사가 아닌 **녹다운** 상태 (기어다니기 가능, 권총만 사용)
- **듀오 모드**: 파트너가 5초 부활 가능
- **솔로 모드**: 녹다운 8초 후 사망

### 9-6. 킬 피드 & 스탯 📊
- 화면 우측 상단에 실시간 킬 피드 (누가 누구를 어떤 무기로 킬했는지)
- 게임 종료 후 결과창:
  - 생존 순위, 킬 수, 피해량, 이동거리
  - MVP 플레이어 하이라이트

### 9-7. 특수 이벤트존 ⚡
- 맵 특정 위치에 **전기 위험지대** 랜덤 생성 (안전구역과 별개)
- 해당 구역 진입 시 초당 8 데미지
- 맵에 경고 표시로 미리 알림

### 9-8. 부스터 아이템 🏃
| 아이템 | 효과 | 지속 |
|---|---|---|
| 아드레날린 | 이동속도 +40% | 8초 |
| 방탄 부스터 | 데미지 -30% | 6초 |
| 저격 부스터 | 이동 중 반동 제거 | 10초 |

## 12. 서버 아키텍처

```
[클라이언트 Phaser3]
        ↕ WebSocket (Colyseus)
[Node.js 게임 서버 - Colyseus]
  ├─ GameRoom (최대 8명/룸)
  ├─ 서버 틱: 20tick/s
  ├─ 서버 권위 (총알 판정, 아이템, 데미지)
  ├─ 클라이언트 보간 처리
  └─ 상태 동기화 (위치, HP, 아이템)
```

- **서버 권위 방식**: 모든 게임 판정은 서버에서 처리 (치팅 방지)
- **위치 보간**: 클라이언트는 서버 상태를 보간하여 부드럽게 렌더링
- **총알 서버 검증**: 클라이언트가 사격 이벤트 전송 → 서버가 히트 판정
